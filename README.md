# infoSnap

# 简介
infoSnap是一个基于Streamlit开发的 AI 碎片化信息管家！一键抓取多平台信息，智能提炼要点，帮你高效筛选、消化信息，构建个人知识库。更有智能搜索、即时问答，让信息回顾触手可及。

## 功能说明
- 提交链接：输入文章链接，系统自动抓取文章内容。
- 文章分析：系统会对文章进行分析，提取关键信息。
- 文章报告：系统会生成一份详细的文章分析报告，包括文章标题、摘要、关键词、关键信息等。

## 依赖列表
- streamlit
- requests
- beautifulsoup4
- openai
